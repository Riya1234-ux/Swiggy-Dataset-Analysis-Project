🍽️ Swiggy Sales Analytics Dashboard
📌 Project Overview
This project presents an interactive Swiggy Sales Analytics Dashboard built using data visualization and business intelligence techniques. The dashboard provides comprehensive insights into sales performance, customer ratings, order behavior, food preferences, and geographical trends across different states and cities.

The goal of this project is to transform raw Swiggy order data into actionable business insights that help stakeholders understand customer purchasing patterns, sales performance, and regional demand.

🎯 Objectives
Analyze overall sales performance.

Monitor customer satisfaction through ratings.

Identify top-performing food categories and locations.

Track sales trends across different time periods.

Support data-driven decision-making using visual analytics.

📊 Key Performance Indicators (KPIs)
The dashboard tracks the following business metrics:

KPI	Description
💰 Total Sales	Total revenue generated from all orders
⭐ Average Rating	Mean customer rating across all orders
🛒 Average Order Value (AOV)	Average revenue generated per order
📈 Ratings Count	Total number of ratings received
📦 Total Orders	Total number of completed orders
📈 Dashboard Visualizations
1. Monthly Sales Trend
Displays sales performance month-wise.

Helps identify seasonal demand patterns and growth trends.

2. Daily Sales Trend
Shows daily fluctuations in sales.

Useful for monitoring short-term performance.

3. Total Sales by Food Type
Compares revenue contribution from different food categories.

Identifies the most popular food segments.

4. Total Sales by State
Visualizes geographical sales distribution.

Highlights high-performing states.

5. Quarterly Performance Summary
Provides quarter-wise sales analysis.

Enables comparison of business performance across quarters.

6. Top 5 Cities by Sales
Identifies cities generating the highest revenue.

Supports location-based business strategy.

7. Weekly Trend Analysis
Analyzes sales behavior across weekdays.

Helps determine peak ordering days.
